Current Projects

This coming semester, we'll be working on the following three projects

Flight - PX4 Firmware and QGroundControl:

The PX4 Flight Management Unit is a low-cost high-end autopilot aimed towards hobbyists and researchers. We have used this hardware last year with the Ardupilot firmware and Mission Planner ground control software (GCS). This year we will be contributing to the open source PX4 Firmware project and the QGroundControl project.

Navigation/Perception - Gesture Control and Visual SLAM:

We are able to control our quadrotors via remote control, joystick, and GCS, but we would also like to add gesture based control using vision sensors (ex. Kinect, PrimeSense). We also aim to work on using these sensors for vision-based Simulataneous Localization and Mapping (SLAM)

Design - Custom Frame+Attachments:

Last year, we used commercial frames and machined custom frames and blade guards for our quadrotors. This year, we will continue this effort and design multicopter frames for various sized UAVs and attachable parts, such as grippers, landing gears, and housings for electronics.

Past Projects

Tour Guide Quads: We worked with the SENSEable City Lab to create a quadrotor and iPhone app capable of giving outdoor tours.

UDB3 Flight Controller: Using a modified UDB3 board and code at the Aerospace Controls Lab, we worked on firmware for a dsPIC30 which allowed for remote controlled flight.

Joystick control: We wrote a Python program using PyGame to allow control of quadrotors via a video game controller.

ROS/MRPT Simulations: We set up 2D and 3D simulation environments and quadrotor models to test navigation/perception software.

Plotting tool: We wrote C# and Python programs for realtime plotting of sensor data.

Blade guards: To prevent our quadrotors from chopping up passersby during flight tests on Kresge Lawn, we designed and machined foam blade guards.

But I want to do something else!

The scope and goals of our three current projects are tentative and will of course, change to better suit the interest of the club members. So if you have a cool idea you want work on, let us know and we'll see if we can add it in! And if you're not sure what you'd like to work on - there's always something to do (like make a better website for example :P).
